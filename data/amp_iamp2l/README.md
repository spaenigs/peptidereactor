# Antimicrobial peptides vs. non-AMPs

##### Description

    The benchmark dataset of [1]. Downloaded from http://www.jci-bioinfo.cn/iAMP/Supp-S1.pdf. Also used in [2], [3], [4],
    [5] and [6].
