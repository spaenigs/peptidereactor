# Neuropeptides vs. non-NEPs

##### Description

    Dataset is taken from [1] and downloaded from https://webs.iiitd.edu.in/raghava/neuropipred/download.php. We 
    used the combined (main + validation) NeuroPIpred_DS1 dataset in this study.
